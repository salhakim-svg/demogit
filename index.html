Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa standar yang digunakan untuk membuat struktur halaman website. Ibarat rumah, HTML adalah fondasi dan dinding-dindingnya.

  • Mudah dipelajari: Struktur kodenya menggunakan tag yang intuitif.
  • Fleksibel: Bisa dikombinasikan dengan CSS (untuk desain) dan JavaScript (untuk interaksi).
